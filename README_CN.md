# 用 38 个 Markdown 文件，把 AI 变成你的私人西语老师

## 一句话说清楚

一套 Markdown 文件，丢进 Gemini CLI 或 Claude Code，AI 就变成一个有记忆、能适应你水平、会苏格拉底式提问的西班牙语家教。不用装 app，不用搭后端，不用写一行代码。

## 这东西解决什么问题

学语言最有效的方式不是背单词、刷题库，是有个好老师跟你一对一对话。

Language Transfer 是一门免费的西语音频课，作者 Mihalis Eleftheriou 用一种叫「思维法」的教学方式，让学生通过提问自己发现语法规则，而不是死记硬背。90 节课，零教材，纯对话。效果好到在 Reddit 上被封神。

问题是——Mihalis 只有一个人，他不能坐在你旁边，根据你的水平和错误实时调整教学。

这个项目做的事很简单：把 Mihalis 的教学方法写成一套 AI 能理解的指令，让大语言模型变成你的私人版 Mihalis。

## 它怎么教你

不讲语法表。不背变位。不说「今天我们学虚拟语气」。

它会这样问你：

```
老师："Preparation" 的西语是 "preparación"。
      把 -tion 去掉，加上 -r……你觉得动词是什么？

学生：Preparar？

老师：对。那 "confirmation" 呢？

学生：Confirmación……所以动词是……confirmar？

老师：以后你再也不会忘这个规律了。
```

一次只问一个问题。等你想完再问下一个。答错了不直接纠正，而是用新的问题引导你自己找到答案。

就像真正的私教在跟你聊天。

## 它记得你是谁

这是跟普通 ChatGPT 对话最大的区别：它有记忆。

每次上课，它会默默记录：
- 你现在什么水平（A1 到 C2）
- 哪些概念你已经掌握了
- 哪些地方你总犯同一个错
- 什么教学方式对你最有效
- 你喜欢聊什么话题（用来造例句）

下次打开终端说「继续上课」，它会从你上次停下的地方接着来。

你甚至可以在课上直接跟它说：「你提示给得太多了」或者「我觉得有点无聊」——它会当场调整，并且永久记住你的偏好。

## 技术上怎么实现的

没有代码。整个项目是 38 个 Markdown 文件。

核心思路：不造轮子，直接用现有的 AI 终端工具（Gemini CLI / Claude Code），通过精心设计的指令文件让 AI 扮演老师。

文件分两类：

**不变的（老师的大脑）**——36 个文件
- 身份、性格、教学方法论
- A1 到 C2 的 27 个知识点文件
- 教学案例、常见错误库、概念地图

**会变的（学生的成长轨迹）**——2-3 个文件
- 学生档案、记忆文件、每次课的笔记

克隆仓库，拿到完整的老师大脑，但学生档案是空白的。AI 会从零评估你，给你建一条专属的学习路径。

## 造这个东西的过程中学到了什么

最反直觉的一件事：造一个教学 AI，最难的不是告诉它该教什么，而是告诉它**什么时候该闭嘴**。

大语言模型天生就想帮你——给你答案、给你解释、给你一堆信息。但好的苏格拉底式老师恰恰相反：他忍住不说，让沉默替他工作。

我们在测试中踩了这些坑：

**AI 忍不住把答案塞进提示里。** 问「怎么说'我想去商店'？」然后括号里写「提示：想=quiero」。这等于直接告诉答案。我们加了规则：永远不在问题里给任何线索。

**AI 一次问三个问题。** LLM 总觉得说得越多越好。我们强制它一次只能问一个问题，问完立刻停。

**AI 陷在一个知识点里出不来。** 开始教虚拟语气之后，连续练了 30 分钟虚拟语气。学生早就走神了。我们加了轮换规则：同一个语法点最多聊 5-6 轮就必须切换话题。

**AI 写字像在写文档。** 用加粗、用列表、用标题格式。我们逼它用纯文字聊天，像一个人在说话，不像一份报告。

**用户关掉终端，记忆全丢。** 所以我们改成课中主动存档，不等下课。即使突然关机也不会白学。

## 谁适合用这个

- 正在学西语，希望有个能记住你进度的AI陪练
- 试过 Duolingo 但觉得太无聊太机械
- 听过 Language Transfer 音频课觉得很棒，但希望有互动版
- 对 AI Agent 设计感兴趣，想看一个纯 Markdown 驱动的 Agent 怎么搭

## 怎么开始

```bash
git clone https://github.com/jarodise/Language-Transfer.git
cd Language-Transfer
gemini  # 或 claude
# 然后说：我们开始上课吧
```

推荐模型：Gemini 3.1 Pro 或 Claude Opus 4.6 效果最好。Gemini 3 Flash 和 Claude Sonnet 4.6 也够用。小模型（Flash Lite、Haiku）会破功——管不住自己的嘴。
