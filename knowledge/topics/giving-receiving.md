# 授受表达：あげる/もらう/くれる

## 这是什么
日语表达"给"和"收"时，必须考虑给予的方向和视角。这在中文里用一个"给"就搞定了，但日语需要根据"谁给谁"选择不同的动词。

verb-forms（动词变形系统）

## 教学方法

### 三个基本动词

**あげる**（我/我方 → 给别人）
- 私は友達にプレゼントをあげた。（我给了朋友礼物）
- 方向：从说话人这边出去 →

**もらう**（我/我方 ← 从别人那里得到）
- 私は友達にプレゼントをもらった。（我从朋友那里收到了礼物）
- 方向：向说话人这边进来 ←

**くれる**（别人 → 给我/我方）
- 友達が私にプレゼントをくれた。（朋友给了我礼物）
- 方向：别人主动给过来，含感激之意 →

引导发现："中文只需要一个'给'。但日语问的是——给的方向是什么？是我给出去，还是别人给我？"

### 关键区别：あげる vs くれる
两个都是"给"，但视角不同：
- あげる = 我（方）给别人（从我方的角度出发）
- くれる = 别人给我（方）（从接收者的角度，带感激）

"想想：如果你是摄像机，あげる是拍'给出去'的镜头，くれる是拍'收到了'的镜头——还带着'谢谢你给我'的温暖滤镜。"

### ～てあげる/～てもらう/～てくれる（为某人做某事）
更实用的形式——て形 + 授受动词：
- 教えてあげる（我教给你/帮你教）
- 教えてもらう（请别人教我/我得到了别人教我的恩惠）
- 教えてくれる（别人教了我——感激）

引导发现："'请帮我拍张照'——你是在请别人为你做事，用もらう还是くれる？"

### 恩惠意识
日语授受表达的核心是"恩惠"意识——做事情是一种恩惠。
- ～てくれてありがとう（谢谢你为我做了这件事）
- 这反映了日本文化中对他人恩惠的敏感。

## 常见陷阱
- あげる和くれる混淆——方向感是关键
- 中文"我给他一本书"直译时，要判断用あげる
- "他给我一本书"直译时，要判断用くれる（不是あげる！）
- もらう的に标记的是"来源"而非"接收者"
