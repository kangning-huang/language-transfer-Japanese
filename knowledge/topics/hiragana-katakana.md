# 平假名与片假名

## 这是什么
日语的两套表音文字系统。平假名用于日语固有词和语法功能，片假名主要用于外来语。中文母语者需要全新学习这两套系统——这是和汉字桥梁互补的另一面。

## 前置条件
无。与汉字桥梁同步进行。

## 教学方法

### 平假名入门
不要一次性甩出五十音图表格。通过使用来教：
- 先教最常用的：あいうえお（元音）、かきくけこ、さしすせそ
- 每教一行，立刻用它造词：あい（爱）、いけ（池）、さけ（酒）
- 让学生发现：平假名其实是从中文草书演变来的
  - あ ← 安（安的草书）
  - か ← 加（加的草书）
  - さ ← 左（左的草书）
  - 这个联系帮助中文母语者记忆

### 片假名入门
等平假名基本掌握后引入：
- 片假名是从中文楷书的偏旁取出来的
  - ア ← 阿（阿的偏旁）
  - カ ← 加（加的偏旁）
  - サ ← 散（散的偏旁）
- 主要用于外来语（英语等）：コーヒー（coffee）、テレビ（TV）、パソコン（personal computer）
- 也用于强调、拟声拟态词

### 教学节奏
- 不要要求一周内背完五十音
- 每次教一行（5个字），在使用中巩固
- 浊音（が行、ざ行等）和拗音（きゃ、しゃ等）在基本假名掌握后引入
- 始终结合实际词汇来教，不做孤立的字符操练

## 常见陷阱
- シ(shi)和ツ(tsu)容易混淆（片假名）
- ソ(so)和ン(n)容易混淆（片假名）
- は作助词读wa不读ha
- を读o（和お同音但用法不同）
- 学生可能急于求成想背完所有假名——提醒他们慢慢来
