# 尊敬语与谦让语体系

## 这是什么
日语敬语的完整体系：尊敬语（抬高对方）和谦让语（压低自己）的系统学习。这是从"会说日语"到"会得体地说日语"的关键跨越。

keigo-intro（敬语入门）

## 教学方法

### 尊敬语的三种构成方式

**方式一：お/ご + ます形 + になる**
- お読みになる（您读）、お帰りになる（您回去）
- ご覧になる（您看）、ご存知（您知道）
- 最通用的尊敬语构成法

**方式二：特殊尊敬动词**（要记忆）
| 普通形 | 尊敬语 | 含义 |
|--------|--------|------|
| いる | いらっしゃる | 在 |
| 行く/来る | いらっしゃる | 去/来 |
| 食べる/飲む | 召し上がる | 吃/喝 |
| 言う | おっしゃる | 说 |
| 見る | ご覧になる | 看 |
| 知っている | ご存知だ | 知道 |
| する | なさる | 做 |
| くれる | くださる | 给（我方） |

**方式三：受身形（被动形作尊敬）**
- 社長が来られた。（社长来了。）—— 用被动形表尊敬
- 先生が話された。（老师说了。）
- 语气比专用尊敬动词稍轻

### 谦让语的两种构成方式

**方式一：お/ご + ます形 + する/いたす**
- お持ちします（我来拿）、お伝えいたします（我来转达）
- ご案内します（我来引导）、ご説明いたします（我来说明）

**方式二：特殊谦让动词**（要记忆）
| 普通形 | 谦让语 | 含义 |
|--------|--------|------|
| いる | おる | 在 |
| 行く/来る | 参る | 去/来 |
| 食べる/飲む | いただく | 吃/喝 |
| 言う | 申す | 说 |
| 見る | 拝見する | 看 |
| 知る | 存じる | 知道 |
| する | いたす | 做 |
| もらう | いただく | 收到 |
| あげる | さしあげる | 给 |

### 实际运用场景
**电话/邮件**：
- "山田さんはいらっしゃいますか。"（山田先生在吗？——尊敬）
- "山田は今おりません。"（山田现在不在。——谦让说自己人）

**介绍第三方**：
- 说自己公司的人：用谦让语（压低自己人）
- 说对方公司的人：用尊敬语（抬高对方）
- 即使是你的社长，对外也要用谦让语！

引导发现："这个很有趣——你的社长在公司里是最大的，但对客户提到社长时，要把他'压低'。因为对外，整个公司都是'我方'。"

### 美化语（丁寧語的延伸）
在名词前加お/ご使其更优雅：
- お水（水）、お茶（茶）、お金（钱）
- ご飯（饭）、お花（花）、お天気（天气）
- 有些已经固化，不加反而奇怪（ご飯、お茶）

## 常见陷阱
- 对自己用尊敬语——最致命的错误（"私がいらっしゃる"是错的！）
- いらっしゃる一词多义（在/去/来）——靠语境判断
- 内外意识：对外说自己公司的人不用尊敬语
- 过度敬语（二重敬語）："お読みになられる"——多了一层，错误
