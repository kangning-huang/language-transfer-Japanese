# 汉字桥梁：中文母语者的最大优势

## 这是什么
利用中文母语者已有的汉字知识，快速建立日语词汇基础。这是整个教学体系中最核心的入口——让学生在第一堂课就意识到他们已经"认识"大量日语。

## 前置条件
无。这应该是最早教的内容之一。

## 教学方法

### 第一步：同形同义词（信心建立器）
先展示完全相同或几乎相同的汉字词：
- 電話（电话）、銀行（银行）、學校→学校、圖書館→図書館
- 経済（经济）、政治（政治）、文化（文化）、歴史（历史）
- 注意（注意）、安全（安全）、問題（问题）

引导发现："你看这些词，在中文里是什么意思？"——学生会惊讶自己认识这么多。

### 第二步：繁简对应规律
日语汉字很多保留了繁体字形：
- 學→学（但日语简化方式不同：学）
- 國→国（日语：国）
- 會→会（日语：会）
- 氣→気（日语独特的简化）
- 圖→図（日语独特的简化）

引导学生发现日语汉字和繁体字的关系："你学过繁体字吗？日语汉字很多和繁体字一样或者很像。"

### 第三步：同形异义词（假朋友）
把这些差异当作有趣的文化故事讲：
- 手紙：中文"手纸/卫生纸" → 日语"信"（手写的纸→书信）
- 勉強：中文"勉强" → 日语"学习"（努力勉为其难地学）
- 大丈夫：中文"大丈夫/男子汉" → 日语"没关系/没问题"
- 娘：中文"妈妈" → 日语"女儿"
- 走る：中文"走" → 日语"跑"（日语的"走く/歩く"才是"走"）
- 汽車：中文"汽车" → 日语"火车"（日语的汽车是"自動車"）
- 新聞：中文"新闻" → 日语"报纸"

### 第四步：音读——连接中文发音
很多音读和中文发音有历史联系：
- 山：中文 shān → 日语音读 さん(san)
- 生：中文 shēng → 日语音读 せい(sei)/しょう(shō)
- 人：中文 rén → 日语音读 じん(jin)/にん(nin)
- 学：中文 xué → 日语音读 がく(gaku)

引导发现："日语的音读其实是从古代中文借来的。虽然经过了上千年的变化，但有些还是能听出相似之处。"

## 常见陷阱
- 学生可能过度自信，以为所有汉字词都和中文意思一样
- 需要逐步引入假朋友，让学生养成确认的习惯
- 不要一次堆砌太多词——每次展示5-8个，让学生消化
