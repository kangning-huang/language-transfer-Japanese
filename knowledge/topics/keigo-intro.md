# 敬语入门

## 这是什么
日语敬语（敬語）体系的入门。日语的敬语远比中文的"您""请"复杂，是一个多层的系统。N4阶段只需要理解基本框架和最常用的敬语形式。

## 前置条件
verb-forms（动词变形系统）

## 教学方法

### 敬语的三层结构
先建立整体概念：
1. **丁寧語**（礼貌语）：です/ます —— 你已经在用了！
2. **尊敬語**（尊敬语）：抬高对方的动作
3. **謙譲語**（谦让语）：压低自己的动作

引导发现："你已经在用敬语了——です和ます就是最基本的敬语。现在我们来看看更高一级的。"

### 丁寧語复习
你已经掌握的：
- ～です、～ます、～ました、～ません
- これは丁寧語。对任何人说都不会失礼。

### 基本尊敬语
抬高对方动作的方式：

**お/ご + 名词/形容词**：
- お名前（您的名字）、お仕事（您的工作）
- ご家族（您的家人）、ご住所（您的地址）
- 规律：和语用お、汉语词用ご

**お + 动词ます形 + になる**：
- お読みになる（您读）、お帰りになる（您回去）

**常用特殊尊敬动词**（先记几个最常见的）：
- いる → いらっしゃる（在/有——尊敬）
- 行く/来る → いらっしゃる（去/来——也是这个！）
- 食べる/飲む → 召し上がる（吃/喝——尊敬）
- 言う → おっしゃる（说——尊敬）

### 基本谦让语
压低自己动作的方式：

**常用特殊谦让动词**（先记几个）：
- いる → おる（在——谦让）
- 行く → 参る/伺う（去——谦让）
- 食べる → いただく（吃——谦让）
- 言う → 申す（说——谦让）
- する → いたす（做——谦让）

引导发现："中文说'我吃了'和'您吃了'用一样的'吃'。日语不一样——说自己吃用'いただく'，说对方吃用'召し上がる'。对方被抬高，自己被压低。"

### 敬语使用场景
- 对上级/长辈/客户：用尊敬语说对方的事+谦让语说自己的事
- 对陌生人/一般场合：丁寧語就够了
- 和朋友：普通形（タメ語）
- 商务/服务场合：最高等级

## 常见陷阱
- 对自己的动作用了尊敬语——这是最常见的错误
- 在同一句话里混用不同等级——保持一致
- 过度使用敬语反而显得生硬——场合不同，程度不同
- 中文母语者容易觉得"太麻烦了"——先从最常用的开始，不需要一次全学会
