# Quiero Basics

## What This Is
"Quiero" (I want) + infinitive structure. First real sentence-building framework.

## Prerequisites
- es-and-no (basic sentence structure)

## Teaching Approach

### Introduction
Hook through something they know: "Te quiero" (I love you / I want you). Remove "te" → quiero = I want. Now we can build.

### Core Structure: quiero + infinitive
- Quiero preparar. (I want to prepare.)
- Quiero cancelar. (I want to cancel.)
- No quiero cancelar. (I don't want to cancel.)
- Quiero explorar. (I want to explore.)

### Verb Derivation (Critical Skill)
Use the -ación → -ar derivation to generate verbs:
- "How is 'cooperation'?" → Cooperación → "And to cooperate?" → Cooperar
- "Celebration?" → Celebración → "To celebrate?" → Celebrar
- "Creation?" → Creación → "To create?" → Crear

Teach the PROCESS: always go through the noun form, never jump straight to the verb.
- Wrong process: "participate" → "participatar" (garbage)
- Right process: "participation" → "participación" → "participar" (correct)

### Adding Pronouns
- **me** = me/myself (same as English, pronounced /meh/)
- **te** = you (informal)
- **lo** = it / him

Attach to infinitive:
- Quiero prepararme. (I want to prepare myself / get ready)
- No quiero obligarte. (I don't want to oblige you)
- Quiero cancelarlo. (I want to cancel it)

### Emphasis with Yo
- "Yo" = I (usually omitted, "quiero" already means "I want")
- Use only for emphasis: "Yo no quiero cancelar." = I DON'T want to cancel.

## Common Pitfalls
- Jumping from English verb to Spanish verb without going through the -ación noun
- Forgetting that "quiero" already contains "I"
- Placing pronouns wrong ("quiero lo cancelar" instead of "quiero cancelarlo")

## Connection to Other Concepts
- Introduces → pronouns-object (me, te, lo expanded)
- Same structure applies to → intento (I try), puedo (I can), voy a (I'm going to)
- Leads into → present-regular (from "quiero + infinitive" to conjugating verbs themselves)
