# Subjunctive Triggers

## What This Is
When to use the subjunctive — the practical guide to which situations demand it.

## Prerequisites
- subjunctive-present

## Teaching Approach

### The Key Rule
"Two different subjects + a trigger word/phrase = subjunctive."
- "Quiero ir" = I want to go (same subject, no subjunctive)
- "Quiero QUE vayas" = I want you to go (different subjects, "que" bridge, subjunctive)

### Category 1: Wishes and Desires (WISHING)
- Quiero que... (I want that...)
- Espero que... (I hope that...)
- Deseo que... (I wish that...)
- Prefiero que... (I prefer that...)
- Ojalá (que)... (I wish / If only...)

Practice: "I want you to come" → Quiero que vengas.

### Category 2: Emotions (FEELING)
- Me alegra que... (I'm glad that...)
- Me molesta que... (It bothers me that...)
- Es triste que... (It's sad that...)
- Me sorprende que... (It surprises me that...)
- Tengo miedo de que... (I'm afraid that...)

### Category 3: Doubt and Denial (DOUBTING)
- Dudo que... (I doubt that...)
- No creo que... (I don't think that...)
- No es verdad que... (It's not true that...)
- Es posible que... (It's possible that...)
- Tal vez / Quizás... (Maybe... — subjunctive when doubt is strong)

Note: "Creo que" (I think) takes INDICATIVE — you believe it's real. "No creo que" takes SUBJUNCTIVE — you doubt it.

### Category 4: Impersonal Expressions (JUDGING)
- Es importante que... (It's important that...)
- Es necesario que... (It's necessary that...)
- Es mejor que... (It's better that...)
- Es raro que... (It's strange that...)

### Category 5: Conjunctions (CONNECTING)
- Para que (so that) — ALWAYS subjunctive
- Antes de que (before) — ALWAYS subjunctive
- Sin que (without) — ALWAYS subjunctive
- Cuando (when) — subjunctive for FUTURE events only
- Aunque (although) — subjunctive if hypothetical

### The "Cuando" Nuance
- "Cuando llega" (when he arrives — habitual, indicative)
- "Cuando llegue" (when he arrives — future, hasn't happened yet, subjunctive)

## Common Pitfalls
- Forgetting subjunctive after "ojalá" (most common trigger error)
- Using subjunctive after "creo que" (takes indicative — you believe it)
- Missing "que" bridge: "Quiero vayas" → needs "Quiero QUE vayas"
- Using indicative with "para que" (always subjunctive, no exceptions)
