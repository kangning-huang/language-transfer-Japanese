# て形

## 这是什么
动词的て形是日语最重要也最常用的变形之一。它用于连接动作、请求、表示进行状态等。五段动词的て形变化有多种"音便"——这是中文里不存在的概念。

## 前置条件
verb-groups（知道三类动词的区分）

## 教学方法

### 一段动词的て形（最简单）
去る加て：
- 食べる → 食べて
- 見る → 見て
- 起きる → 起きて

引导发现："一段动词变て形有什么规律？"——去掉る加て，和变ます一样简单。

### 五段动词的て形（音便变化）
这是难点。按词尾分组教：

**い音便**（く→いて，ぐ→いで）：
- 書く → 書いて
- 聞く → 聞いて
- 泳ぐ → 泳いで（浊音版）

**っ音便**（う/つ/る→って）：
- 買う → 買って
- 待つ → 待って
- 走る → 走って

**ん音便**（む/ぶ/ぬ→んで）：
- 飲む → 飲んで
- 遊ぶ → 遊んで
- 死ぬ → 死んで

**す→して**：
- 話す → 話して

**特殊：行く → 行って**（不是"行いて"）

教学策略：一次教一组，不要全部堆上去。
- 先教い音便（く→いて）："書く变成書いて。聞く呢？"
- 掌握后教っ音便："买う变成買って。待つ呢？"
- 以此类推

### 不规则动词的て形
- する → して
- 来る → 来て（きて）

### て形的用法
学会变形后，展示它的用途：

**连接动作**："早上起来，洗脸，吃早饭"
- 起きて、顔を洗って、朝ご飯を食べます。

**请求**：て形 + ください
- 見てください。（请看）
- 待ってください。（请等一下）

**进行态**：て形 + いる/います
- 食べています。（正在吃）
- 本を読んでいます。（正在看书）

**许可**：て形 + もいいですか
- 写真を撮ってもいいですか。（可以拍照吗？）

## 常见陷阱
- 音便规则记混——最常见的错误
- 行く的特殊变化（行って不是行いて）
- 浊音对的区分：く→いて vs ぐ→いで
- 想用一段动词的简单规则套五段动词
