# 动词变形系统

## 这是什么
在て形基础上，扩展到日语动词的完整变形体系：ない形（否定）、た形（过去）、可能形、意志形、命令形。这些变形组合起来构成日语表达的骨架。

## 前置条件
verb-groups（动词分类）、te-form（て形）

## 教学方法

### ない形（否定）
**一段动词**：去る加ない → 食べない、見ない
**五段动词**：词尾う段→あ段+ない
- 書く → 書かない、飲む → 飲まない、話す → 話さない
- 特殊：う结尾 → わない（買う → 買わない，不是"買あない"）
**不规则**：する → しない、来る → 来ない(こない)

引导发现："て形时，く变成了いて。ない形呢？書く→書かない——く变成了什么段？"

### た形（过去）
好消息：た形的变化规则和て形完全一样，只是て→た、で→だ
- 書く → 書いた、飲む → 飲んだ、食べる → 食べた
- 引导发现："你已经会て形了。把て换成た，就是过去式！"

### 可能形（能做某事）
**一段动词**：去る加られる → 食べられる（能吃）
**五段动词**：词尾う段→え段+る → 書ける（能写）、飲める（能喝）
**不规则**：する → できる、来る → 来られる(こられる)

注意：口语中一段动词常省略ら——食べれる（ら抜き言葉），虽然非标准但很常见。

对中文母语者的联系："中文说'能吃'、'会写'——日语把'能/会'变成了动词本身的变形。"

### 意志形（打算/让我们）
**一段动词**：去る加よう → 食べよう
**五段动词**：词尾う段→おう段 → 書こう、飲もう
**不规则**：する → しよう、来る → 来よう(こよう)

用法：
- 自己的意志："行こう"（我去吧/走吧）
- 邀请："一緒に食べよう"（一起吃吧）

### 命令形与禁止形
**命令形**（很直接，慎用）：
- 一段动词：去る加ろ → 食べろ
- 五段动词：词尾う段→え段 → 書け、飲め
- 用于紧急情况、上级对下级、体育教练等

**禁止形**：辞书形+な
- 行くな（别去）、食べるな（别吃）

## 常见陷阱
- ない形中"う→わない"的特殊变化容易忘
- た形和て形规则相同——利用这点加速学习
- 可能形：一段动词的"られる"和被动形相同，容易混淆
- 意志形和命令形的语气差别——意志形温和，命令形非常直接
