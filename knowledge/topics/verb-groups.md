# 动词分类：五段、一段、不规则

## 这是什么
日语动词分三类，每类的变形规则不同。正确分类是掌握所有动词变形的基础。中文动词不变形，所以动词变形系统对中文母语者来说是全新的。

## 前置条件
desu-masu（知道ます形是什么）

## 教学方法

### 辞书形（原形）和ます形的关系
先建立两种形式的对应：
- 食べる → 食べます
- 飲む → 飲みます
- 行く → 行きます
- する → します
- 来る(くる) → 来ます(きます)

### 三类动词

**一段动词（ru-verbs）**：
- 辞书形以 -える 或 -いる 结尾
- 食べる、見る、起きる、寝る、教える
- 变形简单：去掉る，加ます → 食べ+ます
- "一刀切"——去掉一个る就行

**五段动词（u-verbs）**：
- 辞书形以 う段 结尾（う/く/す/つ/ぬ/ぶ/む/る）
- 飲む、書く、話す、買う、待つ、走る
- 变ます形：词尾う段→い段+ます
  - 飲**む** → 飲**み**ます
  - 書**く** → 書**き**ます
  - 話**す** → 話**し**ます

引导发现："飲む→飲みます，書く→書きます——词尾发生了什么变化？都是从う段变到了い段！"

**注意：以る结尾的陷阱**
有些以-いる/-える结尾的动词其实是五段动词：
- 走る(はしる) → 走ります（五段！）
- 入る(はいる) → 入ります（五段！）
- 帰る(かえる) → 帰ります（五段！）
- 这些需要逐个记忆，但常见的就那几个

**不规则动词**：
- 只有两个！する（做）和来る（来）
- する → します
- 来る(くる) → 来ます(きます)
- 好消息：只需要记这两个特殊的

### 汉字优势：する动词
很多汉字词 + する = 动词：
- 勉強する（学习）、練習する（练习）、料理する（做菜）
- 運動する（运动）、旅行する（旅行）、掃除する（打扫）
- 中文母语者的巨大优势——这些汉字词你都认识！

引导发现："你知道'勉強'是学习。加上する就变成动词'学习'了。那'練習'加する呢？"

## 常见陷阱
- 以る结尾的五段动词误判为一段动词
- 汉字+する动词的する被遗忘
- 五段动词的い段变化不熟练
- 中文母语者最大的挑战：习惯"动词要变形"这件事本身
